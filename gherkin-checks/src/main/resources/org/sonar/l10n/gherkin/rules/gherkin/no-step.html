A Background, a Scenario or a Scenario Outline that does not define any step is useless. It can either be safely removed or steps should be added.

Noncompliant Code Example

Background:

Scenario: My Scenario

Scenario Outline: My Scenario outline

Compliant Solution

Background:
  Given ...

Scenario: My Scenario
  Given ...

Scenario Outline: My Scenario outline
  Given ...
