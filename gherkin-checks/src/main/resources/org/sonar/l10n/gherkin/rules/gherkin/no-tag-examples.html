Tags should be set on Scenario Outline instead of Examples.

Noncompliant Code Example

Scenario Outline: Scenario...
  Given blabla...
  When blabla...
  Then blabla...

  @tag1 @tag2
  Examples:
    | value |
    | 1     |
    | 2     |

Compliant Solution

@tag1 @tag2
Scenario Outline: Scenario...
  Given blabla...
  When blabla...
  Then blabla...

  Examples:
    | value |
    | 1     |
    | 2     |
