It is useless to set a tag that is already defined at Feature level on Scenario, Scenario Outline and Examples.

Noncompliant Code Example

@mytag
Feature: my feature

  @mytag @othertag
  Scenario: my scenario
    ...

  @mytag
  Scenario Outline: my scenario
    ...

    @mytag
    Examples:
      ...

Compliant Solution

@mytag
Feature: my feature

  @othertag
  Scenario: my scenario
    ...

  Scenario Outline: my scenario
    ...

    Examples:
      ...
