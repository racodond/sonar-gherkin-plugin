If a tag is set to all scenarios, it should be set to the feature instead.

Noncompliant Code Example

Feature: My feature

  @mytag
  Scenario: My scenario #1
    ...

  @mytag
  Scenario Outline: My scenario #2
    ...

Compliant Solution

@mytag
Feature: My feature

  Scenario: My scenario #1
    ...

  Scenario Outline: My scenario #2
    ...
